Windows Tools VM¶

Overview¶

This Windows Server 2012 R2 image comes pre-installed with a number of tools, including:

  • Microsoft Remote Server Administration Tools (RSAT)
  • PuTTY, CyberDuck, WinSCP
  • Sublime Text 3, Visual Studio Code
  • OpenOffice
  • Python
  • pgAdmin
  • Chocolatey Package Manager

Deploy this VM on your assigned cluster if directed to do so as part of Lab Setup.

Only deploy the VM once, it does not need to be cleaned up as part of any lab completion.

Deploying Tools VM¶

In Prism Central > select > Virtual Infrastructure > VMs, and click Create VM.

Fill out the following fields:

  • Name - Initials-Windows-ToolsVM
  • Description - (Optional) Description for your VM.
  • vCPU(s) - 1
  • Number of Cores per vCPU - 2
  • Memory - 4 GiB
  • Select + Add New Disk
    • Type - DISK
    • Operation - Clone from Image Service
    • Image - ToolsVM.qcow2
    • Select Add
  • Select Add New NIC
    • VLAN Name - Secondary
    • Select Add

Click Save to create the VM.

Power on the VM.

Login to the VM via RDP or Console session, using the following credentials:

  • Username - NTNXLAB\Administrator
  • password - nutanix/4u
